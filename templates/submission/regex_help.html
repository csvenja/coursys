Regular Expression Help

Regular expressions must use the Python regular expression syntax and are applied to the filename using the .match() (as opposed to .search()) method.

For example, the regular expression you want to match C or C++ code might be one of these (depending whether you care about case or not):

^.*\.(c|cpp)$
^.*\.(c|C|cpp|CPP)$

Or if you have several options for filename (possibly because of options in the assignment) you might have a regular expression like this:

^(Chess|Checkers|Go)\.java$
